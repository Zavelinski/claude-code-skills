geo-aeo-audit for Claude Code

geo-aeo-audit is a Claude Code skill that audits a page for citability by AI answer engines (ChatGPT, Perplexity, Google AI Overviews, Gemini, Copilot) and gives you the exact structure to add so they can quote you. Classic SEO optimizes to rank. GEO/AEO optimizes to be quoted inside the answer.

What it does

It scores the page against a citability rubric (liftable lead, definition block, question-headed FAQ, JSON-LD, entity clarity) and outputs paste-ready structure: a definition block, JSON-LD markup, and an FAQ, scoped to AEO not classic SEO.

Install

Option 1, Claude Code plugin marketplace (recommended):

/plugin marketplace add Zavelinski/claude-code-skills
/plugin install geo-aeo-audit@claude-code-skills

Option 2, manual fallback: clone the source repo and run its installer. The marketplace path registers any hooks through the Claude Code consent UI; the manual installer edits ~/.claude/settings.json directly and is blocked by auto-mode. Prefer Option 1.

How do I use it?

  • Give it a page URL or HTML: /geo-aeo-audit https://example.com/page.
  • Read the citability score and the rubric breakdown.
  • Paste in the definition block, JSON-LD, and FAQ it outputs.

Frequently asked questions

What is the difference between AEO and SEO?
SEO optimizes to rank in the results list. AEO (answer engine optimization) optimizes to be quoted inside the AI answer itself. This skill is scoped to AEO: it makes your page quotable by ChatGPT, Perplexity, and AI Overviews.
What does geo-aeo-audit output?
A citability score, a rubric breakdown, and paste-ready structure: a liftable definition block, JSON-LD markup, and an FAQ with question headings. You paste these into your page.
Does it improve my search ranking?
It targets being quoted by AI answer engines, not classic ranking. Better AI citability can bring referral traffic, but the skill is scoped to AEO and does not make classic SEO claims.
