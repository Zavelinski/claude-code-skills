godmode for Claude Code

godmode is a one-word max-capability mode for Claude Code. Say godmode and Claude runs every task at full power for the rest of the session: it maps and uses the best available tool (skill, plugin, MCP, agent, or workflow) instead of the first one that works, stays exhaustive over fast, verifies before claiming done, and persists context.

What it does

Say godmode once. For the rest of the session Claude (1) maps every relevant capability and picks the best fit per task, (2) is exhaustive and correct over quick and cheap, (3) verifies before it says done, (4) keeps changes small, visible, and reversible, and (5) persists context at the end. Say godmode off to exit.

Install

Option 1, Claude Code plugin marketplace (recommended):

/plugin marketplace add Zavelinski/claude-code-skills
/plugin install godmode@claude-code-skills

Option 2, manual fallback: clone the source repo and run its installer. The marketplace path registers any hooks through the Claude Code consent UI; the manual installer edits ~/.claude/settings.json directly and is blocked by auto-mode. Prefer Option 1.

How do I use it?

  • Turn it on: godmode
  • Work as normal; Claude stays in max-capability mode and announces the tool it chose.
  • Exit: godmode off

Frequently asked questions

What does godmode do in Claude Code?
godmode turns on a max-capability mode for the rest of your Claude Code session. Claude maps all available skills, plugins, MCP connectors, agents, and workflows, picks the best one for each task instead of the first that works, stays exhaustive, verifies before claiming done, and persists context.
Is the godmode hook safe?
Yes. The godmode tracker is a local-only UserPromptSubmit hook with no network access. It injects a behavioral reminder into your prompt context. Install it through the /plugin flow so the hook registers via the Claude Code consent UI, not by manually editing settings.json.
How do I turn godmode off?
Say godmode off in any prompt. The mode is session-scoped, so it also clears when you start a new session.
