weekly-ai-digest for Claude Code

weekly-ai-digest is a Claude Code skill that produces a sourced, skimmable weekly AI roundup. Ask for "what happened in AI this week" and Claude pulls the last 7 days from multiple sources, dedupes and clusters by story, ranks by signal, and writes a link-backed markdown digest. No hype, every factual claim linked.

What it does

It collects the last 7 days from multiple sources, dedupes and clusters stories, ranks them by signal, and writes a markdown digest grouped into Headlines, Models and Releases, Research, Tools, and Discussion. Every factual claim is linked.

Install

Option 1, Claude Code plugin marketplace (recommended):

/plugin marketplace add Zavelinski/claude-code-skills
/plugin install weekly-ai-digest@claude-code-skills

Option 2, manual fallback: clone the source repo and run its installer. The marketplace path registers any hooks through the Claude Code consent UI; the manual installer edits ~/.claude/settings.json directly and is blocked by auto-mode. Prefer Option 1.

How do I use it?

  • Ask: what happened in AI this week.
  • It collects, dedupes, clusters, and ranks the last 7 days.
  • You get a markdown digest with every claim linked.

Frequently asked questions

Where does weekly-ai-digest get its sources?
It pulls the last 7 days from multiple sources, including web search and research connectors when available, dedupes and clusters by story, and ranks by signal.
Is every claim linked?
Yes. Every factual claim in the digest is backed by a link. No hype, no unsourced assertions.
How is the digest structured?
A skimmable markdown grouped into Headlines, Models and Releases, Research, Tools, and Discussion, ranked by signal within each group.
