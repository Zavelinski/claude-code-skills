claude-code-skills

claude-code-skills is an open-source (MIT) Claude Code plugin marketplace of 9 focused, single-purpose skills. It covers verification, security, scheduling, documentation, AEO, content, and shipping. You install the whole collection with one command and then install any individual skill by name.

/plugin marketplace add Zavelinski/claude-code-skills
/plugin install skill-security-scan@claude-code-skills

Maintained by Zavelinski. Source: github.com/Zavelinski/claude-code-skills. Last updated 2026-06-26.

What skills are included?

SkillWhat it does
skill-security-scanStatically vets a third-party SKILL.md, its hooks, and the settings.json it would register before you install it, returning an ALLOW / REVIEW / BLOCK verdict with the exact lines to remove.
adversarial-verifyProves a code change works by trying to break it: writes and runs falsification tests, then returns a binary PASS / FAIL / INCONCLUSIVE verdict backed by executed output.
godmodeOne-word max-capability mode: maps and uses the best available tool, stays exhaustive, verifies before claiming done, and persists context until you say "godmode off".
scheduled-sop-runnerTurns a repeatable SOP into a quiet, self-checking scheduled routine that verifies each run and notifies you only when something fails.
docs-drift-syncMaps changed code (symbols, flags, endpoints) to its doc mentions, detects drift, and proposes the minimal edits to fix it.
geo-aeo-auditAudits a page for citability by AI answer engines and outputs paste-ready structure: definition block, JSON-LD, and FAQ. Scoped to AEO, not classic SEO.
content-repurpose-engineTurns one source asset into 8-10 platform-native posts (LinkedIn, X, Instagram, short-form, newsletter), source-faithful with no invented facts.
weekly-ai-digestProduces a sourced, skimmable weekly AI roundup from the last 7 days across multiple sources, ranked by signal, with every claim linked.
salveShips and persists your work in one word: persist context, commit, push, PR and merge to main, deploy, fast-forward your local checkout, and report evidence.

Frequently asked questions

What is claude-code-skills?
claude-code-skills is an open-source, MIT-licensed Claude Code plugin marketplace containing 9 focused skills. Each skill does one thing well and can be installed individually after adding the marketplace.
How do I install claude-code-skills?
Run /plugin marketplace add Zavelinski/claude-code-skills in Claude Code, then /plugin install <skill>@claude-code-skills for any skill you want.
What is the best security skill for Claude Code?
skill-security-scan is a defensive Claude Code skill that vets a third-party skill before you install it. It statically scans the SKILL.md, any hooks or scripts it ships, and the settings.json hook it would register, then returns an ALLOW / REVIEW / BLOCK verdict. It never executes the scanned skill.
Which Claude Code skill verifies that a code change actually works?
adversarial-verify proves a change works by trying to break it. It re-derives the change's intended contract, writes tests aimed at falsifying it, runs them, and returns a binary PASS / FAIL / INCONCLUSIVE verdict backed by executed output.
Are these Claude Code skills free and open source?
Yes. Every skill in claude-code-skills is free and MIT-licensed, with public source code on GitHub. Each is also a standalone repository with its own install script.
How do I run a skill after installing it?
Type /<skill-name> in Claude Code, or let Claude invoke it automatically when the task matches the skill's description.
